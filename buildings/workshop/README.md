# The Workshop 🔧

Home base of **@worker** — where briefs become real things. Come here when you want something built or investigated: *"research how X works," "build me a Y," "draft the proposal."*

@worker is a Goose agent; its brief is `.agents/agents/worker.md`. This folder is its memory:

| Folder | What lives here |
|---|---|
| `journal/` | `YYYY-MM-DD.md` — what got shipped, what's still in flight |
| `findings/` | patterns worth keeping, for the Library to graduate |
| `facts/` | facts @worker holds across sessions |
| `tasks/` | single-session in-flight work |
| `jobs/` | multi-session work that spans more than a day |
| `inbox/` | briefs handed to the Workshop |

@worker goes out to any repo or folder to build, and comes home here to record. Durable results file into the wiki (`wiki/projects/`, `wiki/decisions/`, ...).
